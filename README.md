# empty-experimental

An experimental graphics engine supporting multiple rendering APIs.

## Supported Graphics APIs:
- OpenGL  
- Vulkan  
- D3D11  
- D3D12  

## Features:
- Cross-platform compatibility  
- High-performance rendering  
- Modular and extensible architecture  
- Support for modern GPU features  

## Goals:
- Provide a lightweight and flexible rendering engine  
- Support multiple graphics APIs with minimal overhead  
- Enable easy integration into existing projects  
- Foster an open and collaborative development environment  

## Status:
- Early development stage  
- Experimental and subject to changes
